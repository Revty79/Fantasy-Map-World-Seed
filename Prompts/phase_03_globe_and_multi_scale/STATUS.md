# Codex Prompt Queue Status — World Seed Mapper — Phase 3

## Phase Goal
Build the authoring layer on top of the completed fractal + heightmap system:
- terrain and biome painting
- water and hydrology editing
- region / feature / symbol placement
- map labels and text tools
- editing utilities needed to make the map actually usable as a worldbuilding canvas

Phase 3 should turn the terrain generator into a real cartography workspace.

## Relationship to earlier phases
- Phase 1 established the flattened-globe-safe world canvas, procedural generation foundation, navigation, and base rendering flow.
- Phase 2 established elevation / terrain adjustment tooling, height editing UX, and terrain-state persistence foundations.
- Phase 3 must build on those systems, not replace them.

## Global rules for this phase
- Do not break the flattened-globe-safe assumptions established in earlier phases.
- Do not introduce globe wrapping yet unless required for preview hooks only.
- All new paint / feature / label systems must remain compatible with later globe conversion.
- Prefer deterministic, data-backed editing over purely visual hacks.
- Keep the editing model layer-based where practical.
- Preserve undo/redo compatibility for all user-authored changes.
- Keep performance acceptable on large world canvases.
